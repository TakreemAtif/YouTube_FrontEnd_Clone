# YouTube Frontend Clone

A frontend clone of the YouTube homepage built using HTML and CSS. This project was developed while following the SuperSimpleDev HTML & CSS course to strengthen core frontend development skills and recreate a real-world user interface.

## Features

- Responsive homepage layout
- Navigation sidebar
- Header with search bar
- Video grid using CSS Grid
- Flexbox-based components
- Organized and maintainable CSS structure

## Technologies

- HTML5
- CSS3
- CSS Grid
- Flexbox

## What I Learned

- Semantic HTML
- Responsive web design
- CSS Grid and Flexbox
- UI implementation from a design reference
- Writing clean and structured CSS
